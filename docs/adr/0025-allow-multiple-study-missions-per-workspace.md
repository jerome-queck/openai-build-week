# Allow multiple study missions per workspace

A Study Workspace will represent a durable course, subject, research area, or shared source collection rather than one individual learning objective. It may contain multiple Study Missions, each organizing one or more related Learning Sessions and Learning Trails around a distinct longer-lived outcome.

The Study Workspace owns the Primary Folder, Managed Assets, Linked Sources, Source Snapshots, and workspace-scoped learner context shared by its missions. A Study Mission references those resources rather than duplicating them. For example, an algebra workspace can contain separate missions for understanding group algebras, completing a tutorial, and studying the Sylow proofs while continuing to use the same notes and textbook.

Dashboard and navigation grouping will therefore follow `Study Workspace -> Study Mission -> Learning Session`. Cross-mission links and Evidence Transfer may connect relevant work without copying its sources.
