# Make the dashboard resume-first

The version-one launch dashboard will prioritize resuming meaningful work over browsing history. Its primary element will be a Resume Card for the most recent resumable unconsolidated Learning Session, showing the current Learning Goal, exact return context, and next suggested action. Resumable sessions and Prerequisite Branches will be grouped under their Study Missions so the learner can recover the structure of ongoing work rather than scan a list of chats.

The universal Session Intake will remain prominent for starting new work. Consolidated Session Outcomes and artifact history remain accessible but visually secondary to active learning. This hierarchy is the version-one baseline to validate through later usability testing rather than a fixed layout for future versions.
