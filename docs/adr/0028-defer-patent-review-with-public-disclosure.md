# Defer patent review while continuing public disclosure

The owner is not pursuing patent protection during the current public beta and accepts that continued public disclosure may limit later patent options; this is not a conclusion that any current feature is patentable or unpatentable. Reconsider professional patent advice before publicly disclosing a future genuinely novel technical mechanism, since copyright, source-available licensing, and trademark protection do not prevent an independent implementation of the same idea or product flow.
