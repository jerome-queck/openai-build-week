# Undergraduate definition item v1

Present this prompt verbatim in each stochastic run:

> Explain uniform continuity for a function between metric spaces, and contrast it with pointwise continuity. Use one boundary-case example.

Expert scoring requires all of the following: the two metric-space domains are named; the order `for every epsilon, there exists delta, for all points` is preserved; delta is independent of the point; and the example is mathematically correct. Any dropped quantifier, domain restriction, or correctness overclaim fails the run.
